# RC 冒烟验证清单（Phase1-Phase3 收敛）

## 1. 构建与安装

1. `:app:compileDebugKotlin` 通过。
2. `:app:testDebugUnitTest` 通过。
3. Debug 包可安装并启动，无冷启动崩溃。

## 2. 核心能力链路

1. 批量记账：一条输入可落多笔，允许部分成功。
2. 时间语义：今天/昨天/前天与聊天回执、账本、编辑页一致。
3. Agent 工具编排：查询/统计/新增/更新/删除可正常执行。
4. 聊天展示：不泄露 JSON 载荷，回执文本可读。

## 3. 稳定性与回归

1. 连续执行 3 轮相同操作，结果口径一致。
2. 删除聊天消息不影响账本交易数据。
3. 设置页 AI 能力中心可正常展示 CRUD + 统计能力说明。

## 4. 发布判定

1. 阻塞缺陷数量为 0。
2. 高风险缺陷数量为 0。
3. 如存在中低风险缺陷，需附带降级方案与修复计划。
