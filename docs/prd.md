# KeepAccounts 产品需求文档 (PRD)

## 1. 产品概述 (Overview)

### 1.1 背景
当前市面上的记账应用大多功能繁杂、操作繁琐，用户难以坚持。本项目旨在打造一款**极简、高颜值、强互动**的 AI 驱动记账 App。通过引入虚拟管家（如 Nanami），结合自然语言对话与语音输入，让记账变得像聊天一样轻松。

### 1.2 核心目标
- **平台**: Android 16 原生应用 (Kotlin + Jetpack Compose)。
- **视觉风格**: 治愈系水彩薄荷绿 (Pastel Healing Style)，结合极致的 iOS 26 磨砂玻璃质感 (Glassmorphism)，高透明度，清晰的层级区分，并统一取消 Emoji 图标，全面使用线条粗细统一（strokeWidth 2.5）的精美 SVG 矢量图标。
- **核心交互**: 语音/文字自然语言输入 -> AI 自动解析提取 -> 本地回执展现。
- **数据安全**: 纯本地存储 (Local-First)，绝不上云，保障用户极大隐私。
- **系统特性**: 深度集成 Android 16 "Rich Ongoing Notifications" (状态栏彩色胶囊)，提供沉浸式体验。

## 2. 用户画像
- **年轻群体**: 追求高颜值 UI 和个性化治愈体验的大学生、职场新人。
- **懒人记账者**: 觉得传统记账软件太复杂，希望“一句话、一段语音搞定”的用户。
- **隐私敏感者**: 纯本地存储，不希望包含自己消费习惯的敏感财务数据上传至云端服务器。

## 3. 核心功能需求

### 3.1 首页仪表盘 (Home)
- **快捷入口**: 顶部突出预算卡片及进度。保留极高显眼度的「💬 跟 Nanami 记账」与「✍️ 手动记一笔」快捷操作按钮。
- **时间轴流水**: 最近动态列表完全按照日期分组（如“今天 03月26日”），并带有单日总收支统计。卡片大字展示分类标签，小字展示备注信息，一目了然。
- **左滑/长按悬浮菜单**: 列表条目支持平滑展开，显示圆润的「修改」和「删除」功能键。全局保证同页面下最多仅一条记录展开。点击修改可预填数据至记账页。

### 3.2 AI 对话记账 (Chat)
- **多模态输入**: 聊天底部输入框左侧驻留 Mic（语音输入）图标，右侧整合图片上传功能。
- **AI 智能识别**: 彻底接入大模型流式 API，理解用户自然语言，在生成治愈系情感回复的同时，静默提取记账 JSON 凭证。
- **沉浸式自定义背景**: 用户可以上传自定义背景图片，系统自动应用柔化与色彩融合算法，保证浅绿色气泡（用户）与纯白色带阴影气泡（AI）的高对比度和可读性。

### 3.3 账本报表管理 (Ledger)
- **时空导航**: 顶部的圆角胶囊 Toggle 切换“月度 / 年度”视图，配合左右方向的 `< >` 快捷导航栏快速切换查阅期间。
- **动态曲线趋势**: 去除生硬柱状图，使用平滑折线曲线图展示特定期间内的支出/收入/结余动态走势，坐标随年月自适应。
- **明细联动**: 下方联动呈现类似首页风格的小分类排行与带每日大标题的明细卡片（自动处理闰年及星期错位）。

### 3.4 专属偏好与我的 (Profile)
重构为四大核心模块（卡片）：
- **核心交互区 (AI 管家设定)**: 设置专属称呼（如“主人”、“宝宝”），支持上传 AI 自定义头像，并且允许切换默认预置性格（如“贴心治愈 Nanami”、“傲娇毒舌 Kuro”、“高冷御姐 Yuki”）。修改后会影响主页UI及全局回复语气。
- **资产与分类**: 编辑并自由编排对应的记账分类、图标。支持拖拽排序设定。
- **数据安全与清理 (完全本地化)**: 取消传统云同步概念，提供“账单导出为 CSV/JSON”与“本地离线导入”以及“一键清理缓存”。
- **个性化皮肤与反馈**: 提供多种治愈系皮肤色系切换（如“水彩薄荷绿”、“樱花粉红”、“天空湛蓝”）。应用具备完整的全局 Compose Theme 绑定逻辑，一键切换全局气泡、按钮、图标背景颜色。

## 4. 专属 Android 16 与系统特性整合
- **状态栏彩色胶囊 (Rich Ongoing Notifications)**:
  在用户退回后台或切出屏幕时，若 AI 恰好正在后端流式处理，或正在录制超长语音，应用将在 Android 16 的系统状态栏（类似灵动岛）展示跳动的彩色微章（如：“Nanami 正在翻开账本...”），并支持点击一键唤回 App。
- **震动触觉回馈 (Vibration/Haptic)**: 对高频操作（手动分类选取、条目滑动展开）进行像素级的 UI 配合极速原生震动反馈。

## 5. 迭代计划
- **v1.0 (MVP)**: 完成 Compose 高保真 UI 搭建（基于最新定稿的细描边 SVG 替代图标方案与全量毛玻璃效果）、Room 数据库落地与双向流控架构。
- **v1.1**: 无状态架构切入，对接大模型（如 SiliconFlow）实现打字机级别的对话记账、本地回执卡片拦截截取。
- **v1.2**: Android 16 特性融入：Rich Ongoing Notifications 和 Android 原生 SpeechRecognizer 语音转文字集成。
- **v1.3**: 高级设定模块完善：主题全局切换动态响应，相册背景融合支持，本地化 CSV 日志出入等。
