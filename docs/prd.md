# KeepAccounts 产品需求文档 (PRD)

## 1. 产品概述 (Overview)

### 1.1 背景
当前市面上的记账应用大多功能繁杂、操作繁琐，用户难以坚持。本项目旨在打造一款**极简、高颜值、强互动**的 AI 驱动记账 App。通过引入虚拟管家（如 Nanami），结合自然语言对话与可解释回执确认，让记账变得像聊天一样轻松。

### 1.2 核心目标
- **平台**: Android 16 原生应用 (Kotlin + Jetpack Compose)。
- **视觉风格**: 治愈系水彩薄荷绿 (Pastel Healing Style)，结合极致的 iOS 26 磨砂玻璃质感 (Glassmorphism)，高透明度，清晰的层级区分，并统一取消 Emoji 图标，全面使用线条粗细统一（strokeWidth 2.5）的精美 SVG 矢量图标。
- **核心交互**: 文字自然语言输入 -> AI 自动解析提取 -> 回执确认/一键纠错 -> 入账。
- **数据安全**: 默认纯本地存储 (Local-First)；后续提供用户授权后的可选登录云同步。
- **系统特性**: 当前已落地本地每日提醒通知；Android 16 "Rich Ongoing Notifications" 为规划项。

## 2. 用户画像
- **年轻群体**: 追求高颜值 UI 和个性化治愈体验的大学生、职场新人。
- **懒人记账者**: 觉得传统记账软件太复杂，希望“一句话就搞定”的用户。
- **隐私敏感者**: 纯本地存储，不希望包含自己消费习惯的敏感财务数据上传至云端服务器。

## 3. 核心功能需求

### 3.1 首页仪表盘 (Home)
- **快捷入口**: 顶部突出预算卡片及进度。保留极高显眼度的「💬 跟 Nanami 记账」与「✍️ 手动记一笔」快捷操作按钮。
- **时间轴流水**: 最近动态列表完全按照日期分组（如“今天 03月26日”），并带有单日总收支统计。卡片大字展示分类标签，小字展示备注信息，一目了然。
- **左滑/长按悬浮菜单**: 列表条目支持平滑展开，显示圆润的「修改」和「删除」功能键。全局保证同页面下最多仅一条记录展开。点击修改可预填数据至记账页。

### 3.2 AI 对话记账 (Chat)
- **输入方式**: 当前版本支持文本输入 + 流式对话；语音能力优先复用系统输入法，不单独开发应用内 Mic。图片/OCR 记账为后续可选能力。
- **AI 智能识别**: 彻底接入大模型流式 API，理解用户自然语言，在生成治愈系情感回复的同时，静默提取记账 JSON 凭证。
- **AI 人性化交互**: 对模糊输入进行最小打扰澄清（如“是午饭 35 元吗？”），并在回执卡片提供一键纠错与回滚入口。
- **沉浸式自定义背景**: 支持上传并本地持久化聊天背景；消息气泡支持时间戳显示，回执时间与账本记录时间保持一致。

### 3.3 账本报表管理 (Ledger)
- **时空导航**: 顶部的圆角胶囊 Toggle 切换“月度 / 年度”视图，配合左右方向的 `< >` 快捷导航栏快速切换查阅期间。
- **动态曲线趋势**: 去除生硬柱状图，使用平滑折线曲线图展示特定期间内的支出/收入/结余动态走势，坐标随年月自适应。
- **明细联动**: 下方联动呈现类似首页风格的小分类排行与带每日大标题的明细卡片（自动处理闰年及星期错位）。

### 3.4 专属偏好与我的 (Profile)
重构为四大核心模块（卡片）：
- **核心交互区 (AI 管家设定)**: 设置专属称呼（如“主人”、“宝宝”），支持上传 AI 自定义头像，并且允许切换默认预置性格（如“贴心治愈 Nanami”、“傲娇毒舌 Kuro”、“高冷御姐 Yuki”）。修改后会影响主页UI及全局回复语气。
- **资产与分类**: 当前支持分类新增/删除并与手动记账联动；分类拖拽排序与图标编辑待实现。
- **数据安全与迁移**: 当前已提供缓存清理入口；后续优先探索账号登录（含微信绑定）与可选云同步，导入导出作为备选离线迁移通道。
- **个性化皮肤与反馈**: 提供多种治愈系皮肤色系切换（如“水彩薄荷绿”、“樱花粉红”、“天空湛蓝”）。应用具备完整的全局 Compose Theme 绑定逻辑，一键切换全局气泡、按钮、图标背景颜色。

## 4. 专属 Android 16 与系统特性整合（现状 + 规划）
- **已实现：每日提醒通知链路**
  通过 `AlarmManager + BroadcastReceiver + NotificationChannel` 实现每日定时提醒与开机后自动恢复。
- **状态栏彩色胶囊 (Rich Ongoing Notifications)**:
  在用户退回后台或切出屏幕时，若 AI 恰好正在后端流式处理，应用将在 Android 16 的系统状态栏（类似灵动岛）展示跳动的彩色微章（如：“Nanami 正在翻开账本...”），并支持点击一键唤回 App。
- **震动触觉回馈 (Vibration/Haptic)**: 对高频操作（手动分类选取、条目滑动展开）进行像素级的 UI 配合极速原生震动反馈。

> 注：本节除“每日提醒通知链路”外，其余为规划项，未在当前版本完全落地。

## 5. 迭代计划
- **v1.0（已完成）**: Compose 主体页面、Room 数据库、DataStore 配置、基础手动记账闭环。
- **v1.1（已完成）**: SiliconFlow 流式对话接入、AI 回执入库、搜索与首页/账本联动。
- **v1.2（进行中）**: 聊天时间语义链路（recordTime）与回执时间统一；账本按月/年过滤与当前月趋势截断。
- **v1.3（下一优先级）**: 核心操作极简化（低认知成本）、分类管理增强（图标编辑/排序）、AI 人性化交互 V1（澄清/确认/纠错）。
- **v1.4（后续）**: AI 智能建议 V2、账号登录与微信绑定预研、可选云同步 PoC。
- **v2.0（规划）**: 可选云同步正式化、图片/OCR 记账、导入导出备份通道与 Rich Ongoing Notifications 按价值推进。
