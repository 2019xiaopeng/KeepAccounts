# KeepAccounts v1.1.0 Release Notes

## Release Title
v1.1.0 · Chat & Ledger Experience Refinement

## 发布日期
2026-03-29

## 版本亮点

### 1) AI 对话体验更自然、更可靠
- 修复“发送后列表不自动回到底部”的问题，消息发出即自动可见。
- 优化 AI 流式分气泡策略：避免生硬断句，连发体验更像真实对话。
- 修复回执日期与账本日期不一致问题，聊天回执与账本/编辑页日期保持一致。

### 2) 记账语义一致性增强
- 修复 AI 回执在“收入”场景下仍显示负号的问题：
  - 收入显示 `+金额`
  - 支出显示 `-金额`
- 聊天回执“修改”入口会继承正确的收支类型与日期，减少二次校对。
- 相对日期（如“昨天/前天”）的记录与展示链路进一步统一。

### 3) 账本页交互与布局稳定性提升
- 账本明细改为每页最多 5 条，并使用翻页查看后续记录。
- 明细卡片区域高度固定为 5 条记录位高，不随当前页记录数量变化。
- 优化药丸切换组件（如“日历记账/统计报表”“月度/年度”等）垂直贴边与居中细节。

## 体验优化汇总（v1.1.0 周期）
- 聊天输入与键盘联动更稳定，减少抖动与错位。
- 非聊天页面的安全区与底部留白策略统一，避免内容被遮挡。
- 设置/子页面顶部安全区滚动表现优化。

## 兼容性与迁移说明
- 本版本无破坏性 API 变更。
- 现有用户数据可直接升级使用。

## 已知说明
- 当前发布不包含导入/导出完整能力（仍按既有路线迭代）。
