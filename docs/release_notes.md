# 猫猫记账 Release Notes

## v1.0.0（首发版）
发布日期：2026-03-28

### ✨ 核心功能
- AI 对话记账：支持自然语言输入并生成记账回执。
- 手动记账：支持分类、备注、金额与时间管理。
- 账本视图：提供日历记账与统计报表两种模式。
- 搜索能力：支持按时间、金额、分类、备注等多条件检索。
- 个性化设置：支持主题、头像、首页 slogan、AI 管家风格配置。

### 🎨 体验优化
- 对话输入区支持多行输入，发送按钮布局优化。
- 账本分段切换交互稳定性提升，减少回流时视觉跳变。
- 首次初始化流程优化：
  - 称呼输入改为明确占位提示。
  - 新增每月预算初始化。
  - 初始化头像上传改为本地持久化存储，降低 URI 失效风险。
- 个人页“已坚持记账 X 天”改为真实数据计算。

### 🚀 启动与更新
- 启动页采用官方 Android SplashScreen 方案，避免二次闪屏。
- 已接入版本检测：启动时检测 GitHub 最新 Release，发现新版本后非强制提示。
- 更新检测策略收敛：加入检查间隔与同版本提示去重。

### 🛠️ 反馈与支持
- 帮助与反馈中心已接入 GitHub Issues：
  - 查看问题列表
  - 提交问题反馈
  - 提交功能建议

### 📌 已知说明
- 导入/导出入口目前为占位交互，后续版本会补齐真实文件读写流程。
- 部分视觉动效为稳定性优先策略，后续可按反馈继续微调。
